### Avaliação

__2 BIMESTRES__

Notas B1 = ```(P1 x 0.3) + (O1 x 0.7)```

Notas B2 = ```(P2 x 0.3) + (O2 x 0.7)```

Final = ```(B1 x 0.4) + (B2 x 0.6)```

+++

### Avaliação

<br>
Aprovado = ```Final >= 7.0```

Reprovado = ```Final < 4.0```

Exame = ```Final >= 4.0 && Final < 7.0```

+++
